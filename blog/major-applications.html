Major Applications of AWS:

Amazon Web Services offers 3 main enterprise IT Applications:

Is a managed desktop computing service

- Allows customers to easily provide cloud-based desktops that include applications and resources to their clients that they can access with any device of their choice, including laptops, iPad, Kindle Fire, Android tablets, etc.

- This service is available at half the cost of most virtual desktop infrastructure solutions

Amazon WorkMail

- Is a secure, managed business email and calendaring service

- Allows users easy access to their email, contacts, and calendars on Microsoft Outlook or any web browser or iOS and Android email applications.

- Can be integrated with existing corporate directories and can be control both the keys that encrypt data and the location in which the data is stored.

Amazon Work Docs

- Is a secure, fully managed enterprise storage and sharing service with strong administrative controls and feedback capabilities that improves user productivity.

- Allows users to comment on files, send them to others for feedback and upload new versions without having to resort to emailing multiple versions of their files as attachments

- This service is accessible to customers through any device and offers IT administrators the option of integrating with existing corporate directories, flexible sharing policies, audit logs, and control of the location where data is stored.
